# ReconX
ReconX es una herramienta OSINT que permite buscar y recopilar información pública de usuarios en múltiples plataformas de forma rápida y sencilla. Diseñada para fines de investigación y ciberseguridad, facilita la identificación de perfiles usando datos abiertos.  ⚠️ Uso solo con fines legales y éticos.
